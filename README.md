<h1>Path Visualizer</h1>
<p>A visual implementation of various algorithms to find the best path between two points in a grid</p>
<p>The algorithms include:</p>
<ul>
  <li>Dijkstra</li>
  <li>A* Search</li>
  <li>Breadth First Search</li>
  <li>Depth First Search</li>
</ul>
